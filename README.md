Movie App
=========

프로젝트 소개 
----------

이 프로젝트는 TMDB 영화 API를 활용하였으며, react 환경에서 영화 리스트 및 상세 화면을 조회 할 수 있는 간단한 SPA입니다.
react 개발 환경에 대한 이해와 기술 습득을 위한 목적으로 작업되었고,
사용한 패키지 등은 아래와 같습니다.

* redux
* react-router-dom
* material-ui
* axios
* immutable or immer
* prop-types
* react-slick
* medium-zoom
* styled-component
